The Sigma Discord Bot

When first joining the Sigma Discord, our bot should message you some instructions on how to join a Sigma Guild. Follow these instructions and a Recruiter will be with you very soon to get you placed into a guild.

Commands Available to You

  • Help
    Command
    Description
    Usage
    !modules
    Lists all bot modules.
    !modules
    !commands
    List all of the bot's commands from a certain module. You can either specify the full name or only the first few letters of the module name.
    !commands Administration
    !help
    Either shows a help for a single command, or DMs you help link if no arguments are specified.
    !h
    !hgit
    Generates the commandlist.md file. Bot owner only
    !hgit
    !readme
    Sends a readme and a guide links to the channel.
    !readme
    !donate
    Instructions for helping the project financially.
    !donate

About the Sigma Alliance

Our aim is to provide a range of guilds under one roof for players all over the world, casual and hardcore and at any level. We create a fun an interactive group where you can grow as a player and make friends at the same time.
